图中显示通过 WAN 链路线性连接的三个路由器。R1 通过 WAN 串行链路连接到 R2，它在以太网接口上有一个 LAN。R2 通过 WAN 串行链路连接到 R1 和 R3，在以太网接口上有一个 LAN。R3 通过 WAN 串行链路连接到 R2，它在以太网接口上有两个 LAN。R1 具有通往互联网的 WAN 串行链路，路由器 R3 具有通往另一个云网络的 WAN 串行链路。图 2 显示来自命令 show i.p. route pipe begin Gateway 的输出。这显示路由器 R-1 的路由表，该路由器有三条直连路由，这些路由具有三个接口 IP 地址，该路由器通过 RIP 获知五条路由，一条候选默认路由以及配置的一个最后选用网关。
