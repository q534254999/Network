一级路由

1 级路由是指子网掩码等于或小于网络地址有类掩码的路由。因此，1 级路由可以是：

  • 网络路由 - 是指子网掩码等于有类掩码的路由。

  • 超网路由 - 是指掩码小于有类掩码（例如，汇总地址）的网络地址。

  • 默认路由 - 是指地址为 0.0.0.0/0 的静态路由。

1 级路由的来源可以是直连网络、静态路由或动态路由协议。
