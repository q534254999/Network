图 1 显示用于配置接口描述的命令。该命令从全局配置模式开始。使用第一个命令可进入接口配置模式，它为 interface gigabitethernet 0/0。下一命令配置该描述，这是 LAN 1 链接描述。字词描述后的文本成为该描述。图中的下一命令在接口上分配 IP 地址，这是 IP 地址：192.168.10.1 255.255.255.0。使用 no shutdown 命令开启此接口。图 2 配置 Gigabit Ethernet 0/0 端口的描述和 IP 地址。该命令以全局配置模式开始。使用第一个命令可进入接口配置模式，它为 interface gigabitethernet 0/1。下一命令配置该描述，这是 LAN 2 链接描述。图中的下一命令在接口上分配 IP 地址，这是 IP 地址：192.168.11.1 255.255.255.0。使用 no shutdown 命令开启此接口。图 3 配置串行接口并以全局配置模式开始。使用第一个命令可进入接口配置模式，它为 interface serial 0/0/0。下一命令配置该描述，它为 description Link to R2。图中的下一命令在接口上分配 IP 地址，这是 IP 地址：209.165.200.255 255.255.255.252。下一命令在串行接口上配置时钟速率，该命令是 clock rate 128,000。使用 no shutdown 命令开启此接口。图 4 是允许学生配置 IP 地址、添加描述以及激活路由器上的接口的互动练习。
