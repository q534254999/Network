图 1 显示用于将描述添加到接口并为其分配 IP 地址的命令。图中显示路由器的命令行界面并以全局配置模式开始。使用第一个命令可进入所需接口，该命令是 interface gigabitEthernet 0/0。下一命令将描述置于接口的配置文件中，此处的命令是 LAN 1 链路描述。下一命令分配 IP 地址和子网掩码，这是 IP 地址：192.168.10.1 255.255.255.0。现在必须使用 no shutdown 命令开启此接口。然后，该接口更改为启动状态。图 2 配置下一个接口。使用第一个命令可进入该接口，该命令是 interface gigabitEthernet 0/1。下一命令将描述置于接口的配置文件中，此处的命令是 LAN 2 链路描述。下一命令分配 IP 地址和子网掩码，这是 IP 地址：192.168.11.1 255.255.255.0。现在必须使用 no shutdown 命令开启此接口。该接口更改为启动状态。图 3 配置下一个接口。使用第一个命令可进入该接口，该命令是 interface serial 0/0/0。下一命令将描述置于接口的配置文件中，此处的命令是 description Link to R1。下一命令分配 IP 地址和子网掩码，这是 IP 地址：209.165.200.255 255.255.255.0。接下来，需要为串行链路分配时钟速率，该命令是 clock rate 128000。现在必须使用 no shutdown 命令开启此接口。该接口更改为启动状态。图 4 显示 show ip route route | Gateway 命令的输出。这会显示路由表并应用过滤器以显示文件中在词 Gateway 后（包括关键字 Gateway）出现的所有内容。表中显示由于前面三个图中的三个已配置接口预计存在的六个条目。图 5 是允许学生配置路由器上的三个接口以便它与前面四个图中的路由器连接的互动媒体元素。
