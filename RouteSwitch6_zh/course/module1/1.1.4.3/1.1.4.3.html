过滤 show 命令输出

默认情况下，生成多页输出的命令在显示出 24 行后会暂停。在暂停输出的结尾处，将会显示“--更多--”文本。按 Enter 键显示下一行，按空格键显示下一组输出。使用 terminal length 命令指定要显示的线路数量。零值 (0) 可以防止路由器在输出屏幕之间暂停。

另一个能改善命令行界面 (CLI) 用户体验的有用功能是过滤 show 输出。过滤命令可用于显示输出的特定部分。要启用过滤命令，请在 show 命令后面输入管道 (|) 字符，然后输入一个过滤参数和过滤表达式。

在管道后面可以配置的过滤参数包括：

  • section - 显示从过滤表达式开始的整个部分

  • include - 包括符合过滤表达式的所有输出行

  • exclude - 排除符合过滤表达式的所有输出行

  • begin - 从符合过滤表达式的行开始，显示从某个点开始的所有输出行

注：可以结合使用输出过滤器和任意 show 命令。

图 1 显示随思科 IOS 使用过滤命令的示例。在本示例中，输入了 show running/config | section line vty 命令，结果为针对 xty 线路运行配置部分。图 2 显示使用过滤命令的示例，它为 show ip interface brief | include up。该输出列出已启动的三个接口，意味着其处于活动状态。图 3 显示使用过滤命令的示例，它为 show ip interface brief | exclude unassigned。该输出列出已启动的三个接口，意味着其处于活动状态。很明显缺少路由器上当前未分配 IP 地址第四个接口，因此，此接口地址未分配且输出中忽略。图 4 显示使用过滤命令的示例，它为 show ip route | begin Gateway。输出列出 show ip route 命令的内容，从关键字 gateway 开始直到输出的结尾。图 5 是允许学生练习使用过滤命令的互动媒体元素。
