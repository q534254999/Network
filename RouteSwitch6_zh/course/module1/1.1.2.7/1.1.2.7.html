在交换机上启用 IP

网络基础架构设备需要 IP 地址才能启用远程管理。通过设备的 IP 地址，网络管理员可以使用 Telnet、SSH、HTTP 或 HTTPS 远程连接设备。

交换机没有可以为其分配 IP 地址的专用接口。而是在称为交换虚拟接口 (SVI) 的虚拟接口上配置 IP 地址信息。

例如，在图 1 中，为第 2 层交换机 S1 上的 SVI 分配了 IP 地址 192.168.10.2/24 和默认网关 192.168.10.1。

使用图 2 中的语法检查器配置第 2 层交换机 S2。

图 1 显示包含两个路由器的 LAN，并且每个路由器都在其两个千兆以太网端口中的每一个上具有一个交换机。全部的四个交换机都连接计算机。图中显示用于在 SVI 上配置 IP 地址的命令，图中的流程配置一个交换机的 IP 地址和默认网关。第一个命令在配置模式下运行并且是接口 VLAN 1，这会使交换机进入接口配置模式。下一命令是 IP 地址 192.168.10.2 255.255.255.0。现在必须使用 no shutdown 命令开启此接口。接下来设置默认网关，然后从全局配置模式配置。该图的命令是 ip default-gateway 192.168.10.1。图 2 是允许学生在交换机上配置 IP 地址和默认网关的互动练习。
