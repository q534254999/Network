卸载许可证

要清除思科 1900 系列、2900 系列和 3900 系列路由器上已激活的永久许可证，请执行以下步骤：

步骤 1.禁用技术包。

  • 使用以下命令禁用已激活的许可证：

Router(config)# license boot module 模块名称 技术包 包名称 disable

  • 使用 reload 命令重新加载路由器。需要重新加载才能禁用软件包。

步骤 2.清除许可证。

  • 清除许可证存储中的技术包许可证。

Router# license clear feature-name

  • 清除用于禁用已激活许可证的 license boot module 命令：

Router(config)# no license boot module 模块名称 技术包 包名称 disable

注：有些许可证不能清除，例如内置许可证。只能移除通过使用 license install 命令添加的许可证。不能删除评估许可证。
