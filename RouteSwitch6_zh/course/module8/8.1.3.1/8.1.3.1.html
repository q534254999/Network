将路由器配置为 DHCPv4 客户端

有时，必须将小型办公室/家庭办公室 (SOHO) 中的思科路由器和分支站点以与客户端计算机类似的方式配置为 DHCPv4 客户端。所用方法取决于 ISP。但是，最简单的配置是使用以太网接口来连接电缆或 DSL 调制解调器。要将以太网接口配置为 DHCP 客户端，请使用 ip address dhcp 接口配置模式命令。

在图 1 中，假设已将 ISP 配置为可为选定客户提供 209.165.201.0/27 网络范围内的 IP 地址。在 G0/1 接口配置了 ip address dhcp 命令后，show ip interface g0/1 命令确认该接口处于活动状态，且地址由 DHCPv4 服务器分配。

使用图 2 中的语法检查器配置连接到 ISP 的接口以从 DHCP 服务器获取地址。

图 1 显示了配置路由器通过 DHCP 获取 IP 地址的命令。该命令在接口配置模式下运行，该示例在全局配置模式下开始。显示的第一个命令，是 interface g 0/1。下一条命令是 ip address dhcp。该图然后显示了命令 show ip interface g 0/1 的输出。此命令列出 IP 地址以及指出地址是由 DHCP 确定的行。图 2 是允许学生将路由器配置为 DHCP 客户端的一个互动练习。
