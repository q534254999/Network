课程概述

正如课程标题所述，本课程的重点是学习网络基础知识。本课程中介绍的实用技能和概念性技能将为您奠定理解基本网络的基础。您将执行以下操作：

  • 研究人际交流与网络通信之间的异同

  • 了解用于规划和实施网络的两个主要模型：OSI 和 TCP/IP

  • 了解网络的“分层”方法

  • 详细研究 OSI 和 TCP/IP 层的功能和服务

  • 熟悉各种网络设备和网络编址方案

  • 了解通过网络传送数据所用的介质类型

本课程结束时，您将能够建立简单 LAN、对路由器和交换机执行基本配置并能实施 IP 编址方案。
