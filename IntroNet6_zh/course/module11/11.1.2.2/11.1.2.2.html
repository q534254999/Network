常见协议

不管在小型还是大型网络中，技术人员的大部分工作都是与网络协议有关的。在小型网络中，网络协议将支持员工所使用的应用程序和服务。常见网络协议如图所示。单击每台服务器可查看简短说明。

这些网络协议将构成网络专家的基本工具集。每种网络协议都会定义：

  • 通信会话任意一端的流程

  • 消息类型

  • 消息语法

  • 信息性字段的意义

  • 消息发送方式和预期响应

  • 与下一层的交互

许多公司都已制定政策，尽可能使用这些协议的安全版本。这些协议是 HTTPS、SFTP 和 SSH。

The figure is an interactive activity that shows several servers connected to a switch, which is connected to a router. When you click each server a callout shows a description of what the server does. The servers include DNS, Telnet, EMail, DHCP, Web, and FTP.
