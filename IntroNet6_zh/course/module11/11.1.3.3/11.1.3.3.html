员工网络使用率

除了要了解流量的变化趋势，网络管理员还必须知道网络的使用是如何变化的。如图所示，小型网络的网络管理员能够不断当场获取很大一部分员工的应用程序使用率 IT“快照”。这些快照通常包含以下信息：

  • 操作系统和操作系统版本

  • 非网络应用程序

  • 网络应用程序

  • CPU 使用率

  • 驱动程序使用率

  • 内存使用率

在一段时间内记录小型网络中的员工快照有助于为网络管理员提供有关协议需求及相关流量变化的信息。这种资源利用率的转变就可能要求网络管理员相应地调整网络资源分配。

Figure shows the Processes tab in Windows task manager and examples of processes running in the windows operating system. There are four buttons on the right side of the image. Clicking the first button highlights an application listed on the Processes tab. Clicking the second button highlights a service listed on the Processes tab. Clicking the third button highlights a system operation listed on the Processes tab. Clicking the fourth button highlights a single program listed on the Processes tab that is running several times and each within its own process.
