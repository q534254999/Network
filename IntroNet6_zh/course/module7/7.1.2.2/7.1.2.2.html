子网掩码

如图 1 所示，在向主机分配 IPv4 配置时，必须配置三个点分十进制 IPv4 地址：

  • IPv4 地址 – 主机唯一的 IPv4 地址

  • 子网掩码- 用于标识 IPv4 地址的网络部分/主机部分

  • 默认网关 – 确定到达远程网络所需的本地网关（例如本地路由器接口 IPv4 地址）

当把 IPv4 地址分配给一台设备时，该设备使用子网掩码来确定自身所属的网络地址。网络地址代表同一网络中的所有设备的地址。

图 2 为点分十进制地址和 32 位子网掩码。注意子网掩码本质上是一个 1 位序列后接 0 位序列的序列。

为了确定 IPv4 地址的网络部分和主机部分，要将子网掩码与 IPv4 地址进行从左到右逐位比较（如图 3 所示）。子网掩码中的 1 代表网络部分，而 0 则代表主机部分。注意，子网掩码实际上不包含 IPv4 地址的网络部分或主机部分，它仅通知计算机如何在给定的 IPv4 地址中查找这些部分。

用于确定网络部分和主机部分的实际流程叫做 AND 运算。

Figure 1 shows the Windows IPv4 Properties window with the IP address, subnet mask, and default gateway address statically configured. Figure 2 shows the dotted decimal format for the subnet mask 255.255.255.0 and the binary format below it. The octets are divided between the third and fourth octet. Figure 3 adds the IP address 192.168.10.10 above the subnet mask from Figure 2. The subnet mask is what determines which bits are network and which bits are host.
