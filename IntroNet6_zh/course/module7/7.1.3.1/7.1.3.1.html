为主机分配静态 IPv4 地址

设备的 IP 地址可以静态分配也可以动态分配。

在网络中，一些设备需要固定的 IP 地址。例如，打印机、服务器和要求固定 IP 地址的网络设备。由于这个原因，通常为这些设备分配静态 IP 地址。

如图所示，也可以为主机配置静态的 IPv4 地址。为主机分配静态的 IP 地址在小型网络中是可行的。但是，在大型网络中的每台主机上输入静态地址非常耗时。必须维护一份准确的清单，列出为每台设备分配的静态 IP 地址。
